### Описание
Бот для автоматической рыбалки. Анализирует полоску мини-игры на экране и удерживает нужную клавишу (`A` / `D`) чтобы держать маркер в зелёной зоне.

### Разрешение экрана
Скрипт настроен под разрешение **1920×1080**.  
Если у тебя другое разрешение — скорректируй координаты полоски в начале файла:
```python

BAR_X2     = 1312  # правый край полоски
BAR_Y      = 55    # верхняя граница
BAR_HEIGHT = 20    # высота захвата
```
Используй `F7` для определения координат и цветов под курсором, `F8` для скриншота полоски.
